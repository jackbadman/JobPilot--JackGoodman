# Job Pilot  
_A desktop job tracking application built with the MERN stack._


## Project Overview

**Job Pilot** is a desktop application designed to support job seekers in efficiently managing their job search process.  

In a market where applicants often apply to hundreds of positions, maintaining accurate records of applications, deadlines, and statuses can become a major challenge.  

**Job Pilot** addresses this by providing an intuitive and user-friendly platform for recording and tracking applications, offering CRUD functionality, automated reminders, and data persistence.  

Unlike other “all-in-one” job tools that prioritise feature quantity over usability, **Job Pilot** focuses on **simplicity, reliability, and user experience** — helping users stay organised and informed without unnecessary complexity.
